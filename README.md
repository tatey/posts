# Posts

An example RubyMotion app using Storyboarding.

## Usage

[Install RubyMotion](http://www.rubymotion.com/developer-center/guides/getting-started/).

``` shell
git clone https://github.com/tatey/posts.git
cd posts
rake simulator
```
